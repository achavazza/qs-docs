Input append

An input with some element in the same row

Example

$ Search

    <div class="input">
        <label>Label</label>
        <div class="input-append">
            <a href="#" class="add-on label">$</a>

            <a href="#" class="add-on btn btn-primary">Search</a>
        </div>
    </div>
